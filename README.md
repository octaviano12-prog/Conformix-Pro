# Conformix Pro

Demo comercial estatica do Conformix Pro, um sistema para gestao de NRs, certificados, ASO, EPI, equipamentos, obras, documentos tecnicos e conformidade industrial.

## Como usar

- Abra `index.html` para a pagina comercial.
- Clique em `Entrar na demo` para acessar `demo.html`.
- Todos os dados exibidos sao ficticios e servem apenas para demonstracao.

## Hospedagem Hostinger Node.js

Este projeto esta pronto para Hostinger Node.js.

Configuracao sugerida:

- Repositorio: `octaviano12-prog/Conformix-Pro`
- Branch: `main`
- Versao Node: `22.x` ou superior
- Comando de construcao: nenhum
- Gerenciador de pacotes: `npm`
- Arquivo de entrada: `server.js`
- Diretorio raiz: `./`

O comando `npm start` roda `node server.js`.

## Modulos demonstrados

- Dashboard comercial.
- Funcionarios ficticios.
- NRs / Treinamentos.
- Certificados com status.
- Carteirinha NR frente e verso com QR Code.
- ASO, EPI, equipamentos, guindastes/munck.
- Clientes, obras, ART/APR/Rigging e matriz de competencia.
- Relatorios para impressao/PDF.
