# Conformix Pro

Demo comercial estatica do Conformix Pro, um sistema para gestao de NRs, certificados, ASO, EPI, equipamentos, obras, documentos tecnicos e conformidade industrial.

## Como usar

- Abra `index.html` para a pagina comercial.
- Clique em `Entrar na demo` para acessar `demo.html`.
- Todos os dados exibidos sao ficticios e servem apenas para demonstracao.

## Hospedagem

Este projeto e estatico. Pode ser publicado em GitHub Pages, Hostinger ou qualquer hospedagem que sirva HTML, CSS, JS e imagens.

## Modulos demonstrados

- Dashboard comercial.
- Funcionarios ficticios.
- Certificados com status.
- Carteirinha NR frente e verso com QR Code.
- Relatorios para impressao/PDF.
